ANALYSE TECHNIQUE & FONDAMENTALE

Deux grandes écoles s'affrontent pour prédire les mouvements de prix. Le trader professionnel les combine.

AT.01Supports, Résistances & Tendances
▼

Les supports sont des niveaux de prix où la demande est suffisamment forte pour stopper la baisse. Les résistances sont des niveaux où l'offre stoppe la hausse. Une résistance cassée devient support (et vice versa).

Tendances (Dow Theory)
HAUSSIÈRE (Uptrend) = Sommets + Hauts + Creux + Hauts successifs
BAISSIÈRE (Downtrend) = Sommets + Bas + Creux + Bas successifs
RANGE = Prix coincé entre un support et une résistance horizontaux
🎯 Setup de trading classique

Le prix teste 3 fois la résistance à 150$ sur une action, puis casse au-dessus avec un fort volume. → Entrée LONG à 150.50$, Stop Loss à 148$, Target: 158$ (résistance suivante). Risk/Reward = 1.5$ risqué / 7.5$ visé = ratio 1:5 → excellent.

AT.02Indicateurs Techniques Essentiels
▼

RSI (Relative Strength Index)

Oscille 0-100. <30 = survente (potentiellement haussier). >70 = surachat (potentiellement baissier). Divergences RSI/prix = signal puissant.

MACD

Moving Average Convergence Divergence. Croisement MACD/Signal = signal. Histogramme positif = momentum haussier. Divergences = retournement potentiel.

Moyennes Mobiles

MA 20 (court terme), MA 50 (moyen), MA 200 (long). Golden Cross (MA50 croise MA200 à la hausse) = signal achat. Death Cross = signal vente.

Bandes de Bollinger

MA + 2 écarts-types. Prix proche bande haute = surachat. Prix proche bande basse = survente. Squeeze (bandes resserrées) = explosion de volatilité imminente.

Volume & OBV

Le volume confirme les mouvements. Cassure avec fort volume = fiable. Cassure avec faible volume = signal faible. OBV = volume cumulatif = indicateur de pression achat/vente.

Fibonacci Retracement

Niveaux 23.6%, 38.2%, 50%, 61.8% (golden ratio), 78.6%. Zones de retracement probables lors d'une correction après une forte tendance.

L'analyse fondamentale consiste à évaluer la valeur intrinsèque d'une entreprise via ses données financières.

Fear & Greed Index

Indicateur CNN 0-100. 0-25 = Fear extrême (souvent bon point d'entrée historiquement). 75-100 = Greed extrême (souvent risque de correction). "Soyez avide quand les autres ont peur" — Buffett.

🎰

Put/Call Ratio

Volume de puts / volume de calls. Ratio élevé = sentiment baissier dominant. Ratio faible = euphorie. Utilisé comme indicateur contrarian : un ratio très élevé peut signaler un retournement haussier.

📊

Short Interest / COT Report

Le Short Interest mesure le % d'actions vendues à découvert. Le rapport COT (Commitment of Traders) montre les positions des gros acteurs sur les futures. Données hebdomadaires publiées par la CFTC.
